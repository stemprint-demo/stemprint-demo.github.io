# StemPrint Demo

Demo webpage repository for the StemPrint paper.

This repository is currently prepared as a review-stage demo workspace. Author
details and final release links can be added after acceptance.

## Deploy

After editing local files, publish the demo page with:

```sh
./deploy.sh "Update demo page"
```

The script stages all local changes, creates a commit, and pushes `main` to the
`pages` remote.
